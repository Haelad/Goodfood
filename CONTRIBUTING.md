## Как внести изменения

1. Форкните репозиторий
2. Клонируйте к себе: `git clone https://github.com/Haelad/Goodfood.git`
3. Создайте ветку: `git checkout -b feature/название-фичи`
4. Сделайте изменения
5. Коммитьте изменения: `git commit -am "Добавил новую функцию"`
6. Пушьте ветку: `git push origin feature/название-фичи`
7. Создайте Pull Request
